# Codeblock Markmap

Renders an interactive zoomable mind map (requires the Markmap library bundled with Breadcrumbs).

## Example — Johnny Decimal as mind map

````breadcrumbs
type: markmap
start-note: "04-johnny-decimal/10 Life Admin.md"
fields: [down]
depth: [3]
title: "Life Admin"
````


```breadcrumbs
type: markmap
dataview-from: '"04-johnny-decimal"'
fields: [down]
depth: [3]
```
